💻 本地工具（推荐开发者）

需要安装Node.js，但功能更强大：

安装Node.js
下载地址：https://nodejs.org/
安装混淆工具
npm install -g javascript-obfuscator # 或者 npm install -g uglify-js # 或者 npm install -g terser
使用工具
# JavaScript Obfuscator javascript-obfuscator input.js --output output.js # UglifyJS uglifyjs input.js -o output.min.js -c -m # Terser terser input.js -o output.min.js -c -m

💡 使用建议

  • 开发环境：使用原始代码，便于调试
  • 测试环境：轻度混淆，测试兼容性
  • 生产环境：高度混淆，保护代码
  • 性能敏感：使用UglifyJS或Terser
  • 安全敏感：使用JavaScript Obfuscator
